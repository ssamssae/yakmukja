## 설명 (Description, 4000자 이내)

약먹자는 매일 같은 시간에 약·영양제·비타민을 드셔야 하는 분들을 위한 가장 간단한 복약 알림 앱입니다.

■ 주요 기능
• 간편 등록 — 휠 피커로 복용량·시간 5초 입력
• 매일 반복 알림 — 등록한 시간마다 "○○ ○알 드실 시간입니다" 푸시
• 복용 체크 — 탭 한 번으로 오늘 복용 완료 표시
• 실시간 카운트다운 — 다음 복용까지 남은 시간 홈에 표시
• 시간대별 그룹 — 새벽 / 아침 / 오후 / 저녁 자동 분류
• 완료 배너 — 오늘 먹을 약을 모두 드셨을 때 응원 메시지

■ 이런 분께 추천
- 아침·저녁 혈압약, 당뇨약 챙겨야 하는 분
- 종합비타민·오메가3 매일 드시는 분
- 가족 약 챙겨주시는 분
- 알림 앱 여러 개 설치하기 번거로운 분

■ 디자인 철학
- 눈이 편한 다크 테마
- 군더더기 없는 한 화면 구성
- 광고 없음, 계정 가입 없음, 데이터 서버 전송 없음

■ 개인정보 보호
모든 약 정보와 복용 기록은 **기기 내부에만 저장**됩니다. 서버로 전송되는 데이터가 없으며 외부 계정도 필요 없습니다.
